  • BroadcastExtension

    A class responsible for handling a broadcast session with the Bandyer platform.

    The BroadcastExtension public interface resemble a RPBroadcastSampleHandler subclass, you should indeed call each BroadcastExtension method in the corresponding method from your RPBroadcastSampleHandler subclass. You cannot instantiate this class directly, you must use the singleton instance provided instead.
